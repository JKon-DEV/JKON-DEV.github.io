Climbing Companion

Minimal climbing utilities: fingerboard timers and climb tracking with full CRUD functionality.

Download APK Source
Android Only• Offline-first• v1.0-beta

Screenshots

Tap to view

What it does

  • Custom fingerboard routines (sets / reps / rests)
  • Track climbs with name, grade, notes and date
  • Save fingerboard presets and reuse favourites
  • Full CRUD (Create, Read, Update, Delete) functionality for climbs/timers
  • Fully offline-first experience

Install

  1. Download the APK
  2. Open it on Android
  3. Allow “Unknown apps” if prompted

Tip: If you want fewer warnings, install from GitHub Releases (signed builds).
